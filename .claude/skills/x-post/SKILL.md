# X Post Workflow

Create X posts in multiple variants for the user to select and refine.

## Workflow

Make a todo list for all tasks in this workflow and work through them one by one.

### 1. Get Brief
If not already provided in the user's message, ask:
1. Topic — what do you want to say?
2. Tone: informational / personal / opinion / promotional
3. Format constraints: single post, thread, reply, or quote tweet?

If the user's message already contains all of this, skip the question and go directly to writing.

### 2. Generate 3 Variants
Write 3 posts with distinct approaches:
- **A — Direct:** leads with the key point, no buildup
- **B — Hook-first:** opens with a question, surprising fact, or bold claim to earn the read
- **C — Personal/narrative:** starts from a personal observation or story angle

Rules for all variants:
- ≤ 280 characters. Show character count for each.
- No unnecessary hashtags unless the user asked for them
- No filler phrases ("In today's world...", "It's important to note...")

### 3. Refine
Wait for user selection. Offer to:
- Adjust tone or length
- Add/remove hashtags
- Expand into a thread (if post contains a point worth elaborating)

### 4. Finalize
Present the final post clearly, formatted and ready to copy.

## Wrap Up
State the final post with character count. Note any thread continuation if created.
